# 快速参考指南

## 数据结构变化

### 旧格式
```json
{
  "name": "某个提示词P",
  "note": "备注信息",
  "text": "prompt content..."
}
```

### 新格式
```json
{
  "name": "某个提示词",
  "direction": "正向",      // 新增：正向/反向/无
  "type": "质量",            // 新增：8种分类之一
  "note": "备注信息",
  "text": "prompt content..."
}
```

## 类型分类说明

| 类型 | 字体颜色 | 使用场景 | 示例 |
|------|--------|--------|------|
| 质量 | 紫色 | 画质、清晰度、细节 | masterpiece, best quality |
| 风格 | 青色 | 风格、流派、艺术形式 | photorealistic, cel shaded |
| 质感 | 粉红色 | 材质、手感、表面质地 | soft skin, silk texture |
| 环境 | 青绿色 | 背景、场景、光线 | indoor, sunlight, night |
| 动作 | 琥珀色 | 姿态、动作、表情 | jumping, smiling, standing |
| 表情 | 橙色 | 面部表情、情感 | happy, sad, confused |
| 着装 | 蓝色 | 衣着、服装、配饰 | dress, hat, jewelry |
| 其它 | 灰色 | 不适合上述分类的项 | misc, custom |

## 方向说明

| 方向 | 背景色 | 使用场景 |
|------|-------|--------|
| 正向 | 绿色 | 想要的特征（inclusion prompts） |
| 反向 | 红色 | 不想要的特征（negative prompts） |
| 无 | 蓝色 | 中立项目，加入时默认作为正向 |

## 快捷操作

### 添加提示词
1. 填写信息
2. 选择方向和类型
3. 点击"添加"

### 搜索提示词
- **按文本**：在搜索框输入关键词
- **按类型**：使用下拉菜单筛选
- **组合**：同时使用两种方式

### 加入生成器
- **"加入"**：自动判断方向智能分配
- **"加入正向"**：强制加入正向区域
- **"加入负向"**：强制加入负向区域

## 主题适配

深色/浅色模式下颜色自动调整，确保：
- ✓ 充足的对比度
- ✓ 清晰的可视性
- ✓ 舒适的观看体验

## 常见场景

### 场景1：管理 AI 绘画提示词
1. 添加"masterpiece"，设为 质量-正向
2. 添加"deformed"，设为 其它-反向
3. 添加"outdoors"，设为 环境-正向
4. 使用"加入"按钮自动分配

### 场景2：按风格查找
1. 在类型筛选选择"风格"
2. 所有风格相关的提示词被过滤显示
3. 从中选择需要的项

### 场景3：编辑已有提示词
1. 点击要编辑的项
2. 点击"编辑选中"
3. 修改方向、类型或内容
4. 点击"确认编辑"
