1. **克隆或下载到 ComfyUI 自定义节点目录**

## 💻 使用指南

### 管理提示词

1. **添加新提示词**
   - 在左侧"新增提示词"区域输入：
     - **名称**：提示词的简短标识（如"风景"、"人物"）
     - **方向**：选择"正向""反向"或"无"（自动添加时使用）
     - **类型**：从 8 个类别中选择（质量、风格、质感、环境、动作、表情、着装、其它）
     - **备注**：可选的描述信息
     - **提示词文本**：完整的提示词内容
   - 点击"✚ 添加"按钮保存

2. **搜索和筛选提示词**
   - **搜索框**：输入关键词快速查找
   - **搜索模式**：
     - 模糊搜索：匹配包含关键词的提示词名称
     - 精确搜索：精确匹配提示词名称
   - **类型筛选**：按提示词类别过滤显示

3. **编辑提示词**
   - 在列表中选中要编辑的提示词
   - 点击"✏️ 编辑选中"按钮
   - 在表单中修改相关信息
   - 点击"✓ 确认编辑"保存更改

4. **删除提示词**
   - 在列表中选中要删除的提示词
   - 点击"🗑️ 删除选中"按钮
   - 确认删除操作

5. **显示细节模式**
   - 勾选"显示细节"复选框
   - 在列表中查看每个提示词的方向、类型、备注等详细信息

### 生成提示词

1. **组合提示词**
   - 从左侧提示词列表选中一条提示词
   - 点击对应的按钮添加到生成器：
     - **➕ 加入**：自动判断方向，添加到对应区域
     - **➕ 加入正向 (P)**：添加到正向提示词区域
     - **➖ 加入负向 (N)**：添加到负向提示词区域
   - 提示词文本会自动附加到右侧对应的文本框

2. **使用快捷键**
   - 选中提示词后按 **`P` 键**：快速添加正向提示词
   - 选中提示词后按 **`N` 键**：快速添加负向提示词

3. **手动编辑**
   - 直接在正向/负向文本框中编辑内容
   - 支持复制、删除等标准编辑操作
   - 可复制生成的提示词到 ComfyUI 中使用

### LLM 提示词生成

1. **点击"🤖 LLM 提示词生成"按钮**
2. **输入您的需求**（自然语言描述）
   - 例如："一个穿着红色连衣裙的女性，微笑，在花园里"
3. **点击"⚡ 生成"按钮**
   - 系统会生成一个标准化的提示词模板
   - 模板包含完整的 8 条规则和可用的提示词参考
4. **点击"📋 复制"按钮**复制生成的内容

### 界面设置

- **语言切换**：点击左上角"🌐"图标选择中文或英文
- **主题切换**：点击左上角"🎨"图标在亮色/暗色主题间切换
- **设置保存**：语言和主题设置自动保存到浏览器本地存储

## 🔧 API 端点

| 方法 | 端点 | 说明 |
|------|------|------|
| POST | `/prompt_manage/get` | 获取所有提示词 |
| POST | `/prompt_manage/save` | 保存所有提示词 |
| POST | `/prompt_manage/add` | 添加新提示词 |
| POST | `/prompt_manage/delete` | 删除指定提示词 |

### 请求/响应示例

**添加提示词**
```json
POST /prompt_manage/add
{
    "name": "风景",
    "direction": "无",
    "type": "环境",
    "text": "beautiful landscape, mountains, clear sky",
    "note": "自然风景提示词"
}
```

**删除提示词**
```json
POST /prompt_manage/delete
{
    "index": 0
}
```

## 📁 数据格式

提示词保存为 JSON 数组格式：

```json
[
    {
        "name": "提示词名称",
        "direction": "正向|反向|无",
        "type": "质量|风格|质感|环境|动作|表情|着装|其它",
        "text": "提示词内容...",
        "note": "备注信息（可选）"
    }
]
```

## 🐛 常见问题

**Q: 提示词没有保存？**
A: 检查 `data` 文件夹是否存在且有写入权限。插件会自动创建。

**Q: 访问时出现 404？**
A: 确保 ComfyUI 服务正在运行，检查是否访问了正确的地址 `http://localhost:8188/prompt_manage_web/`

**Q: 使用方法在哪里？**
A: 点击界面右上角的"📹 使用方法"按钮即可观看。

**Q: 如何导入/导出提示词？**
A: 直接编辑 `data/prompts.json` 文件或通过 API 端点进行批量操作。
